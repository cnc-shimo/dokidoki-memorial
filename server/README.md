# RESTful API

## Run
 * docker-compose up -d

## Scale
 * docker-compose scale cakephp-web=5

## Endpoint
 * [GET] /api/v1/users
 * [GET] /api/v1/users/:id
 * [GET] /api/v1/users/:user_id/frustrations
 * [GET] /api/v1/couples
 * [GET] /api/v1/couples/:id
 * [POST] /api/v1/users/:user_id/frustrations
