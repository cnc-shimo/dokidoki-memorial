# どきどきメモリアル
どきどきメモリアルは Android App でカップルや夫婦間のすれ違いを減らす目的で開発しました. 使い方は簡単で日頃の小さな不満や照れくさくて伝えられていない感謝を日記につけます. すると, 相手にはバクダンの大きさで不満のたまり具合が可視化されます. バクダンが大きくなってきたら日記を振り返り話し合ってみましょう. 愛情いっぱい手作りケーキは忘れずに.
